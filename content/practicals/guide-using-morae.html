HCI Practicals: Using Morae

Morae was a product of TechSmith for Windows and used to be found at morae.com, but unfortunately was discontinued on 31 Dec 2019 and is no longer supported. The final version was Morae 3.3.4. The software is still available for download at the Internet Archive:

  • https://web.archive.org/web/20200830113913/https://download.techsmith.com/morae/enu/334/Morae_Manager_Full.exe
  • https://web.archive.org/web/20200830113913/https://download.techsmith.com/morae/enu/334/Morae_Recorder.msi
  • https://web.archive.org/web/20200830113913/https://download.techsmith.com/morae/enu/334/Morae_Observer.msi

The user manuals for Morae 3.3.0 are available here:

  • https://assets.techsmith.com/Docs/pdf-morae/morae-manager-330-user-guide.pdf
  • https://assets.techsmith.com/Docs/pdf-morae/morae-recorder-330-user-guide.pdf
  • https://assets.techsmith.com/Docs/pdf-morae/morae-observer-330-user-guide.pdf

You would need to install Morae Recorder to record a session and Morae Manager to edit findings and produce a highlights video. Morae Recorder does not require an installation key. Morae Manager starts as a 30-day free trial, which works great and is fully featured. It does not introduce watermarks into the videos. However, it is no longer possible to purchase a licence key.

  • I have prepared a Morae configuration file, which you might want to use. [Right-click, Save Link As...]

  • Do not use Morae's facility to present tasks to the user. Use task slips printed out on paper, which the facilitator should read out to the user, and then place on the table next to the computer.

  • Do not use Morae's survey facility to present questionnaires to the user. Use paper questionnaires. The facilitator should control the pen for the background questionnaire. The test user is given the pen for the feedback questionnaire.

  • In the Morae Recorder 3.3.4 configuration, under Main: choose "Screen (Primary Monitor)" as the main video source. Modify the Screen Settings (the wrench icon) to select Default compression the default frame rate (5 frames per second).

    The default codec for screen video in Morae is the TechSmith Screen Capture Codec (TSCC), which is lossless. You can also explicitly select the codec under Custom (advanced). If it is not already installed, you can download the TSCC codec for Windows from here.

    Under PiP: choose the laptop webcam as the PiP video source. Choose the laptop webcam microphone as the audio source.

  • Turn off the task logger.

  • Turn off observers (unless you are familiar with Morae and know what you are doing).

  • Morae records user sessions in .rdg files. Inside the .rdg file, usercam (webcam) and screencam (screen capture) are recorded in separate streams. It is possible to reposition or turn off the usercam window prior to exporting video.

  • In Morae 3.3.4, the standard Morae Begin Task and End Task events are not treated as markers. You have to set your own Start Task (S) and End Task (E) markers in addition (at the same timestamp place), so that these are then exported with all the other markers.

  • Add any task scores (measure of success) to the Morae Begin Task and End Task events, not to your own Start Task (S) and End Task (E) markers.

  • Set markers in Morae for the various events (negative observation, positive observation, quote from user, etc.) which occur during a test. Later, exported the markers for each test user from Morae as a CSV file (Analyze tab → bottom window → select recording → Right-click somewhere in table → Export Results... → CSV file), massage (or convert) them into HTML, and include them in your report to form the test session transcript.

  • When exporting video from Morae, turn off the annoying Morae mouse trail (in Morae Manager under Tools → Manage Plugins → Mouse Path).

  • When exporting video from Morae, choose MP4 as the file format. In Morae Manager, choose the Present tab (on the left side) → right-click on video clip → Produce and Share Clip... → Produce to file → MP4 → Largest video size.

  • You will also need to export the complete session capture of each user to an MP4 file from Morae, for example tp1.mp4, tp2.mp4, etc. In Morae Manager, create a video clip spanning the entire session, then choose the Present tab (on the left side) → right-click on video clip → Produce and Share Clip... → Produce to file → MP4 → Largest video size.
